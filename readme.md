## How to run

- Mac : https://github.com/harryosmar/mock-web-app/blob/master/build/mock-web-app-darwin-64
- Windows 64 Bit : https://github.com/harryosmar/mock-web-app/blob/master/build/mock-web-app-windows-64
- Windows 32 Bit : https://github.com/harryosmar/mock-web-app/blob/master/build/mock-web-app-windows-32

```shell script
git clone git@github.com:harryosmar/mock-web-app.git

cd mock-web-app

# for mac
./build/mock-web-app-darwin-64
```
